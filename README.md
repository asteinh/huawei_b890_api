# huawei_b890_api

## Get started

```python
import huawei_b890_api

# fill your username and password
b890 = huawei_b890_api.HuaweiB890API('user', 'password')

# some exemplary calls
b890.status()
b890.login()
b890.diagnosis()
b890.status()
b890.logout()
```
